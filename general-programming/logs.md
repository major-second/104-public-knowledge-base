# verbosity
- 例如`[0, 10, 20, 30, 40, 50, 'DEBUG', 'INFO', 'WARN', 'ERROR', 'CRITICAL']`
- 设置`DEBUG`也能看到`INFO`，以此类推
- 例如[[nbextension]]中`jupyter contrib nbextension install --Application.log_level=30 --user`
- 设置太高了可能就是[[silent]]，设置太低就太啰嗦
