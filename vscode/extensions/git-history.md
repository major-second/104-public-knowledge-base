# 基于git history插件的git历史管理

- #vscode 安装 #git history 插件后
- 查看一个文件或文件夹的历史记录，
左边文件栏中右键点击某个文件/文件夹
![](git-history1.png)
选择 git: view file history点击即可
![](git-history2.png)
文件编辑者和branch一目了然！
- 查看整个库
  - 法一：`Ctrl+Shift+P`搜索`history`
  - 法二：`Ctrl+Shift+G`出来的那个界面（左侧git图标那个界面）每个库标题右边有个“时钟”图标
    - 这个方便选择你要看[[submodule]]中的哪个
