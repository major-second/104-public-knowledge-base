- 值得注意的
  1. 列表中插入其它元素：使用四个空格或tab！当然如果本身是多级列表，就可能需要更多空白符，比如下面这个
        ```text
        嘿嘿
        ```
  2. 有序无序列表非常好用
     - 而不加`-`或`1.`直接换行可能在[[markdown-preview-enhanced]]中可以换，但其它一些渲染器中不行，属于非标准的行为
