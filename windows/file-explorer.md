# 选择、编辑
- 按一个文件，shift，再按另一个，选中一系列
- ctrl多选
- 这些和[[general-win-editting-keys]]技巧有些可以互相参考
# 查看
- 右键，可以group, sort等
  - 比如常见的：group by type，sort by date，在[[apple-photo]]中有很多live photo时，导出时很有用
  - group之后按相应group可以选中所有
